- Pure HTML/CSS/JS
- Google Fonts (Inter)
- Rose/Coral gradient theme with midnight background
- Dark/Light mode with localStorage persistence
- Full-bleed stats bar, 2x2 philosophy grid, centered about section
- Responsive design
- Intersection Observer animations
